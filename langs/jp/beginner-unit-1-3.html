# 标准日本语 初级（上）1－3 单元总结

# 五十音

不用考虑什么记忆技巧，认的次数多了就记住了。片假名可能需要加强下记忆，可以使用多邻国专门练习。

微软日语输入法
  • Alt+~ 切换英文和日文输入法
  • 促音（っ）的两种输入方式
    • 将后一个音的第一个字母输入两遍，如 学校 がっこう gakkouu
    • 先输入 l 再输入促音
  • F6/F7 转换为平假名/片假名

# 形容词 / 形容动词

* 在新标日教材中称为 一类形容词 / 二类形容词

# 形容词

一般来说以い结尾的都是

いい／よい　悪い　新しい　古い　楽しい　苦しい　易しい　難しい
広い　狭い　熱い　冷たい　高い　安い　高い　低い　
おもしろい　つまらない　すばらしい　若い　優しい　白い　青い　赤い　辛い　甘い　暖かい
  • 直接接名词（楽しい人です）
  • 做谓语　　（楽しいです）
  • 过去式　　（楽しかったです）
  • 否定式　　（楽しくないです／楽しくありません）
  • 过去否定式（楽しくなかったです／楽しくありませんでした）

# 形容动词

一般都为汉字词，不以い结尾，注意加粗的两个例外

きれい（綺麗）　好き　嫌い　上手　下手　便利　不便　静か　にぎやか
安心　不安　簡単　有名　親切　暇　元気　ハンサム　おしゃれ
  • 接名词时要加な（元気な人です）
  • 做谓语　　　　（元気です）
  • 过去式　　　　（元気でした）
  • 否定式　　　　（元気ではないです／元気ではありません）
  • 过去否定式　　（元気ではなかったです／元気ではありませんでした）

# 副词

とても　いつも　よく　ときどき　たまに　あまり　全然（ぜんぜん）
ずっと　ますっぐ　やっぱり　少し　ちょっと　ちょうど　もう　たしか
  • あまり和全然后面须接否定形式（あまり～ません）

# 助词

は　が　に　で　を　と　も　や　から　まで　へ　ね　よ　か　な

  • わたしは料理が上手です／できます／わかります。

  • 李さんは今教室にいます。本は本棚にあります。（表示存在的场所，～在～）
    本棚に本があります。　　　　（表示存在的场所，～有～）
    王さんに電話します。
    家に帰ります。家へ帰ります。（表示移动行为的目的地）
    携帯電話を取りに行きます。　（表示移动行为的目的）
    課長に（から）スケジュール表をもらいました。
    いつも７時に起きます。　　　（表示时间）
    友達に会います。　　　　　　（表示动作的对象）

  • 公園で歌を歌います。美術館で写真展があります。（在某处发生某事用で）
    車で会社へ行きます。バスで行きます。
    新聞紙で紙飛行機を作ります。
    航空便でプレゼントを送ります。
    日本の季節の中で、どの季節がいちばん好きですか。一年で７月がいちばん暑いです。

  • わたしも行きます。　（相当于「也」）
    どこへも行きません。（表示全面否定）
    どれも好きです。　　（🤔でも？）

  • 静かな町　にぎやかな上海　有名な人

# 动词

（いる）います　（ある）あります　行きます　来ます　書きます　買います　見ます　
食べます　飲みます　します　読みます　歩きます　撮ります　取ります　歌います　
遊びます　運転します　働きます　休みます　ジョギングします　旅行します　出張します

# 疑问代词

何（なん／なに）　いつ　どこ　だれ　どれ　どの人　どの国　いくつ

疑问代词读一调

# 文型

  • ～は～です。　　（第二个～可以是名词 / 形容词 / 形容动词）
  • ～に～があります／います。（存在）
    ～は～にあります／います。
  • ～から～まで。　（从～到～，可以是时间或者地点）
  • 李さんは森さんより若いです。　　　（～比～更 形容词 / 形容动词）
    森さんより李さんのほうが若いです。（与～相比～更）
  • 東京の冬は北京の冬ほど寒くないです。　（～不如～，形容词否定形式）
    神戸は大阪ほどにぎやかではありません。（⸺，形容动词否定形式）
  • スポーツ の 中でサッカー が いちばんおもしろいです。（～中～最 形容词 / 形容动词）
