# 标准日本语 初级（上）4－6 单元总结

# 形容词 / 形容动词

形容词て形，い变为くて
形容动词て形，加で

# 助词

  • 小野さんはプールへ泳ぎに行きます。（表示移动行为的目的，目的动词要去掉ます）
    李さんは私の家へ遊びに来ました。
  • 李さんは電車に乗りました。　　（表示人或物体的附着点，不能用を）
    李さんはかばんに書類を入れました。
  • 今年中に結婚したいです。　　　（表示该期间结束之前）
  • 駅前を通ります。橋を渡ります。（经过 / 离开某场所）
    家を出ます。大学を卒業します。
  • このレストランはおいしいですが、ちょっと高いです。　（转折）
  • 東京タワーへ行きたいんですが、どうやって行きますか。（铺垫，多用于书面语；口语用けど）
  • 私は新しい洋服が欲しいです。　（愿望，想要）
  • あの方が山田さんです。（新信息做主语？）
    山田さんはあの方です。
  • 部屋にだれかいますか。（「房间里有人吗？」）
    部屋にだれがいますか。（「房间里有谁？」）
    （疑问词 いつ／どこ／だれ／何／何度 + か，表示某时 / 某地 / 某人等，类似于 somewhere / someone）
  • だれでも分かります。（「都」，肯定形式）
    だれも分かりません。（「都」，否定形式）
  • 小野さんも一緒に行きますよね。（对自己说出的想法没有足够把握时，大部分时候类似于「吧」）
  • みんなで餃子を作ります。（一人で／自分で／全員で）

# 动词て形

🤔

教育文法 vs. 学校文法

一类动词，去掉ます之后最后一个音位于い段的绝大部分动词
二类动词，去掉ます之后最后一个音位于え段，以及位于い段的一小部分动词
三类动词，来ます 和 します，以及使用 します 的动词
（比如前接汉字词或外来词，仕事します，旅行します，コピーします）

……

  • 小野さんは今新聞を読んでいます。（正在进行时，〔动词て形〕+ います）
  • 私は北京に住んでいます。（表示动作的结果状态，〔动词て形〕+ います）
    車を持っていますか。
    ーーいいえ、持っていません。（否定回答）
    田中さんを知っていますか。
    ーーいいえ、知りません。　　（特例）

# 敬体形与简体形

动词 敬体形 简体形
现在将来形式，肯定 買います 買う　　　　（基本形）
现在将来形式，否定 買いません 買わない　　（ない形）
过去形式，肯定 買いました 買った　　　（た形）
过去形式，否定 買いませんでした 買わなかった（なかった形）

* ありません的简体形是ない

形容词（做谓语） 敬体形 简体形
现在将来形式，肯定 忙しいです 忙しい　　（去掉ます即可）
现在将来形式，否定 忙しくないです 忙しくない
过去形式，肯定 忙しかったです 忙しかった
过去形式，否定 忙しくなかったです 忙しくなかった
形容动词（做谓语） 敬体形 简体形
现在将来形式，肯定 簡単です 簡単だ
现在将来形式，否定 簡単ではありません 簡単ではない
过去形式，肯定 簡単でした 簡単だった
过去形式，否定 簡単ではありませんでした 簡単ではなかった
名词（做谓语） 敬体形 简体形
现在将来形式，肯定 晴れです 晴れだ
现在将来形式，否定 晴れではありません 晴れではない
过去形式，肯定 晴れでした 晴れだった
过去形式，否定 晴れではありませんでした 晴れではなかった

# 文型

  • 卵を１個食べます。　　　　　　（数量词用在动词之前）
    本を２冊買いました。
  • 李さんは毎日７時間働きます。　（时间数量 + 动词，小李每天工作 7 小时）
  • 李さんは１週間に２回プールへ行きます。（一周两次，常省略为週に／日に／月に／年に～回）
  • 家へ帰って、宿題をします。　　（动作相继发生，〔动词て形〕+ 动词）
    家へ帰ってから、もう一度日本語の勉強をします。（差别为てから不能在一个句子中多次使用）
  • この荷物を航空便で送ってください。
    （〔动词て形〕+ ください，命令或要求某人做某事，虽然翻译为「请」，但是并不表示敬意）
    この部屋には入らないでください。　（——否定形式，〔动词ない形〕+ で + ください）
  • 家で仕事をしてもいいです。　　（表示许可，〔动词て形〕+ も + いいです）
    ここに座っていいですか。
  • 明日は残業しなくてもいいです。（表示不做某事也行，〔动词ない形〕→なくても + いいです）
  • 教室で物を食べではいけません。（表示禁止，〔动词て形〕+ は + いけません）
  • このリンゴは大きくて、甘くて、おいしいです。
    （形容词 / 形容动词并列使用，用て形，形容词和形容动词可以连接使用）
    森さんは親切で優しいです。
    スミスさんは旅行会社の社員で、営業部の部長です。（名词并列使用，用で连接）
  • （私は）映画を見たいです。　（想（做）～，〔动词去掉ます〕+ たいです）
  • 少し休みませんか。　　　　　（动词否定形式加上疑问句，表示提议）
  • ちょっと休みましょう。　　　（ます→ましょう，表示提议（一起做某事），语气偏随意）
  • 何か食べましょうか。　　　　（——，此外还可表示提议自己为对方做某事）
  • だんだん暖かくなります。　　（表示变化，形容词い变く + なります）
    テレビの音を大きくします。　（因主语的意志而产生变化，+ します）
  • もう元気になりました。　　　（表示变化，形容动词 / 名词 + に + なります）
    来年社会人になります。
    部屋をきれいにしてください。（因主语的意志而产生变化，+ に + します）
    会議室を禁煙にします。
  • 部屋は広いほうがいいです。　（某种性质更好，〔形容词〕+ ほうが + いいです）
    子供は元気なほうがいいです。（——，〔形容动词〕+ な + ほうが + いいです）
  • もっと野菜を食べたほうがいいですよ。　（某种行为更好，〔动词た形〕+ ほうが + いいです）
    そんなに慌てないほうがいいですよ。　　（不～也好，〔动词ない形〕+ ほうが + いいです）
  • 速いですから、飛行機のほうがいいです。（——，〔名词〕+ の + ほうが + いいです）
  • 李さんは今日早く帰らなければなりません。（必须，〔动词ない形→なければ〕+ なりません）
    すぐに書類を送らないと（いけません）。
    （——，主要用于口语，〔动词ない形〕と + いけません（可省略））
  • 李さんはピアノを弾くことができます。
    （表示能力，〔动词基本形〕+ ことが + できます／できません）
    日本では何歳から結婚することができますか。（也可以表示允许 / 可能）
  • 私の趣味は切手を集めることです。（〔动词基本形〕+ こと = 形式名词）
  • 毎日寝る前にシャワーを浴びます。（一个动作在另一个动作之前发生，〔动词基本形〕+ 前に～）
    会議の前に資料をコピーします。　（——，〔名词〕+ の + 前に～）
  • 映画を見た後で、食事をしました。（一个动作在另一个动作之后发生，〔动词た形〕+ 後で、～）
    仕事の後で、食事をしました。　　（——，〔名词〕+ の + 後で、～）
  • 私はすき焼きを食べたことがあります。
    （表示过去的经历，「～过」，通常用于至少半年以前的事情，〔动词た形〕ことがあります）
  • 昨日の試験、どうだった？
    ーーちょっと難しかったけど、まあまあできたよ。
    （けど连接两个句子，表示转折，用于口语；书面语用が）
    来週、送別会をするけど、都合はどうかた？　（或者表示铺垫）
  • 小野さんは休みの日、散歩したり買い物に行ったりします。
    （列举若干种动作，〔动词た形〕→たり）
    飛行機のチケットは高かったり安かったりです。
    （列举多种状况，〔形容词过去简体形～かった→たり）
    この公園は時間によっで静かだったりにぎやかだったりです。
    （——，〔形容动词——〕～だった→たり）
    日本語の先生は中国人だったり日本人だったりです。（——，〔名词——〕，——）
    浴衣は高かったり安かったり、いろいろあります。　（更多例子）
  • この公園は時間によっで静かだったりにぎやかだったりです。
    （「根据……（而不同）/ 因……而异」）
  • この電車はいつも込みますか。
    ーーいいえ、時間によります。（也可以以另一种形式用于句尾）

（疑问句做从句）

  • （私は）今年の夏、北京へ行きますか。＋私は分かりません。
    （不包含疑问词的疑问句（相当于英语的一般疑问句））
    →私は今年の夏、北京へ行くかどうか分かりません。
    （动词 / 形容词变为〔简体形〕然后加上かどうか）
    ＝私は今年の夏、北京へ行くか行かないか分かりません。
    （也可以使用〔基本形〕加〔ない形〕表示相同的含义）
  • キムさんは中国人ですか。＋私は知りません。（——）
    →私はキムさんが中国人かどうか知りません。
    （形容动词 / 名词直接加かどうか，中间不加だ）
    ＝私はキムさんが中国人か中国人ではないか知りません。（かどうか小句的主语后面必须用が）
  • かぎがどこにあるか教えてください。（有疑问词的小句，aka 特殊疑问句）
    昨日何を食べたか忘れました。　　　（使用〔简体形〕加か，形容动词 / 名词同上，不加だ）
    どの料理が辛いか知りません。
  • 李さんはもうすぐ来ると思います。
    （「（我）觉得 / 想～」，～小句用〔简体形〕，只能表示说话人自己的思考内容）
    馬さんは日本へ行きたいと思っています。
    （「～觉得 / 想～」，若是～と思いました或～と思っています则也可以表示其他人的思考内容）
  • 森さんは中国語の試験は難しかったと言いました。　（转述别人的话，～と言いました）
    小野さんは李さんにちょっと休みたいと言いました。（指出话是对谁说的，～に）
  • すみません、頭が痛いんです。
    （〔简体形〕～んです／のです，说明状况或解释原因，两者分别用于口语和书面语）
  • すごい、人気ですね。
    ーーええ、あの歌手は日本ではとでも有名なんです。
    （——，但是名词和形容动词（的现在将来形的肯定形）的だ要换成な）
  • どうして食べないんですか。（询问理由，省略的用法为どうしてですか）
    もうお腹がいっぱいなんです。
  • 日本の音楽についてどう思いますか。（～について，相当于「关于～」）
