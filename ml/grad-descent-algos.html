# 背景

回顾一下，现实中我们有了数据集 D={(x(1),y(1)),⋯,(x(m),y(m))}D = \lbrace(x^{(1)},y^{(1)}),\cdots,(x^{(m)},y^{(m)})\rbraceD={(x(1),y(1)),⋯,(x(m),y(m))}，也定义了损失函数 ℓ ⁣:Y×Y→R\ell\colon\mathcal{Y}\times\mathcal{Y}\to\mathbb{R}ℓ:Y×Y→R，我们想要寻找一个假设函数 hhh，which 能最小化经验误差（hhh 的所有参数由 θ\thetaθ 表示）：

min⁡θf(θ;D),where f(θ;D)=1m∑i=1mℓ(h(x(i);θ),y(i)).\min_\theta f(\theta;D)\text{,}\qquad\text{where } f(\theta;D)=\frac{1}{m}\sum_{i=1}^m \ell\mathopen{}\left(h(x^{(i)};\theta), y^{(i)}\right)\mathclose{}. θmin​f(θ;D),where f(θ;D)=m1​i=1∑m​ℓ(h(x(i);θ),y(i)).

这个优化问题就可以用梯度下降来解决

θ′=θ−η⋅g\theta^\prime = \theta - \eta\cdot g θ′=θ−η⋅g

其中 η\etaη 为学习率 (learning rate)，g=∇E^(θ)g = \nabla\widehat{E}(\theta)g=∇E(θ) 为梯度（下文把目标函数简记为 fff）

  • gt=∇f(θt)g_t = \nabla f(\theta_t)gt​=∇f(θt​)，原始梯度
  • vt≔θt+1−θtv_t \coloneqq \theta_{t+1} - \theta_tvt​:=θt+1​−θt​，实际更新量
